# SignalProcessing
This toolbox contains various signal processing filters and provides an easy to use interface for further expansion.

# Contents
The library currently contains 4 IO and 7 signal processing filters:
+ IO
   - VectorInput
   - RiffWaveInput
   - VectorOutput
   - RiffWaveOutput
+ Signal Processing
   - FrequencyGenerator
   - FrequencyShift
   - AmplitudeModulator
   - FrequencyModulator
   - Fir
   - Differentiator
   - LowPass
